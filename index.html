Informations

  • There can be only one skeleton with only one skin
  • You can continue adding bones by selecting point
  • You cannot draw additional bones if skin was already created
  • You cannot move joints if skin was already created, but you can use forward kinematics
  • There is alternative mode for drawing bones and drawing skin, by holding ctrl.
  • Others button disability depends on selected type (none, joint, bone, conneted bones)
